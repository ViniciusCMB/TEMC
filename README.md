# Tópicos Especiais em Matemática e Computação

Resumo das atividades práticas da disciplina, com notebooks e dados organizados.

## Estrutura

- notebooks/atividades: exercícios curtos (Atividade2, 3, 4, 6, 7, 8, 12, 15)
- notebooks/trabalhos: entregas maiores (T1, T2, T3)
- data/raw: conjuntos de dados usados pelos notebooks

## Principais frentes estudadas

- Estatística descritiva e manipulação de dados com pandas (Atividade2), usando
  séries de admissões do CAGED e operações como sumários, agregações e
  verificações de datas.
- Validação cruzada e modelos básicos de regressão/aprendizado (Atividade8),
  aplicando KFold, regressão linear e RandomForest sobre a base de diabetes.
- Mineração de padrões de cesta de compras (Atividade15), incluindo limpeza da
  base de vendas de outubro/2019 e uso de apriori/association_rules para gerar
  regras de associação.
- Trabalhos T1, T2 e T3 registram aplicações mais completas dos conceitos da
  disciplina em datasets fornecidos.

## Dados utilizados

- data/raw/diabetes.csv: base de diabetes Pima Indians
- data/raw/ipea_admissoes_caged.csv: série de admissões do CAGED
- data/raw/Sales_October_2019.csv: vendas de outubro/2019
- data/raw/blood_transfusion_data.csv: base de transfusão sanguínea (reserva)
